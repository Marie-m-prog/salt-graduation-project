# salt-graduation-project
Graduation project for salt school of applied technology

## Installation

Install all the dependencies in both client and server folder.

```bash
cd server
npm i
```
```bash
cd client
npm i
```

## Usage

Start the client on port 3000 and the server on port 8080.

```bash
npm start
```
